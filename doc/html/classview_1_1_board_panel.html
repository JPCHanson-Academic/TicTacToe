Detailed Description

A BoardPanel object IS a Jpanel, it just specifies a little bit how the JPanel should look. A BoardPanel is a Jpanel populated by buttons(CellButton). Most Work is done in the default constructor(and some private helper methods). It does have a method to set The text of a button.
